# BABYLON Runtime

* Patch your BabylonJS scenes without modifying *.*babylon* file.
* *JQuery*-like API to interact with 3D contents.

## Quick Look
```
[
    {
        "material1" : {
            diffuseColor : "#ff0000",
            diffusePower : 0.1
        }
    }, 
    {
        "mesh2" : {
            position : {
                x : 10
            }
        }
    }
]
```
